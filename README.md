# Python Coding Class 🐍

A collection of **Python practice programs and exercises** created while learning Python programming.  
This repository contains basic Python scripts covering fundamental programming concepts such as strings, lists, functions, loops, and file handling.

The goal of this repository is to **practice Python coding and build a strong foundation in programming concepts**.

---

## 📂 Repository Contents

The repository includes several Python programs that demonstrate important programming concepts:

| File | Description |
|-----|-------------|
| HelloWorld.py | Basic Python program to print output |
| RepeatFunction.py | Demonstrates the use of functions and repetition |
| string1.py | Practice problems related to string operations |
| list1.py | Exercises on Python lists and sorting |
| wordcount.py | Counts word frequency from text using dictionaries |

These programs help in understanding **core Python fundamentals like data structures, loops, and functions**.

## ▶ How to Run the Programs

1. Install Python (Python 3 recommended)

2. Clone the repository

```bash
git clone https://github.com/AlbashX/Python-Coding-Class.git
```

3. Navigate to the project folder

```bash
cd Python-Coding-Class
```

4. Run any Python file

```bash
python filename.py
```

Example:

```bash
python HelloWorld.py
```
